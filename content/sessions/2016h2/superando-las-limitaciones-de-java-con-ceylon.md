El sistema de tipos de Java tiene algunas limitaciones con las cuales hay que lidiar constantemente, a tal grado que los programadores ya ni siquiera las ven como un problema, sin embargo sí afecta el diseño del software que desarrollan. Ceylon arregla muchos de estos problemas con su sistema de tipos, permitiendo dejar atrás algunos de los patrones que a pesar de ser erróneos se usan por no tener alternativas, para poder escribir código más claro y con una intención bien definida.

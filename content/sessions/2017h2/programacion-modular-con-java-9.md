<p><span>La versión 9 de Java, introduce un nuevo tipo de componente de programación: el módulo, que es una colección de códigos y datos autodescriptivos. Este sistema modular introduce una nueva fase opcional: el tiempo de enlace, el cual contiene tiempo de compilación y de ejecución durante el cual un conjunto de módulos puede ser ensamblado y optimizado en una imagen de tiempo de ejecución personalizada, con el uso de la nueva herramienta JLINK.</span></p>
